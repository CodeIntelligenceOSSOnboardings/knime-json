This plug-in provides JSON reading functionality.

This plug-in makes use of the following libraries/projects:

For JSONPath streaming capabilities
jsurfer-core-1.6.0.jar,	https://github.com/jsurfer/JsonSurfer,	MIT License,	https://github.com/jsurfer/JsonSurfer/blob/master/LICENSE,	WANG Lingsong
jsurfer-jackson-1.6.0.jar,	https://github.com/jsurfer/JsonSurfer,	MIT License,	https://github.com/jsurfer/JsonSurfer/blob/master/LICENSE,	WANG Lingsong


Please refer to the libs directory for details on each library
